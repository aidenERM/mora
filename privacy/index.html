Mora Privacy Policy

Mora is an AI assistant developed by AidenERM. This Privacy Policy explains how Mora handles information when users use Mora features and connected services.

Information Access

Mora may access information that users explicitly authorize, including Google services such as Gmail and Google Calendar.

Google API Usage

Google data accessed by Mora is used only to provide requested functionality, including:

  • Email organization and intelligence
  • Calendar assistance
  • Productivity features
  • AI assistant functionality

Data Usage

Mora does not sell user information. Mora does not use Google user data for advertising purposes. Google data is only used to provide features requested by the user.

User Control

Users can manage or remove Google permissions at any time through their Google account settings.

Security

Mora takes reasonable measures to protect user information and maintain responsible data handling practices.
