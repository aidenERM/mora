About Mora

The application name is Mora.

Mora helps users manage information, analyze documents, understand images, and interact with connected services using artificial intelligence technology.

Mora is an independent application created by AidenERM and is not affiliated with Google or any Google product.

Mora Features

  • AI-powered assistance
  • Information organization
  • Document analysis
  • Image understanding
  • Connected service integrations

Google Account Data Usage

Mora requests access to Google services only when a user chooses to connect their Google account and grants permission.

Google API access is used only to provide features requested by the user. Mora does not request Google data for advertising or sell Google user data.

Users remain in control of their account permissions and can revoke access at any time.
